# Snake Game

A classic Snake game built with HTML5 Canvas, CSS, and JavaScript.

## Features

-   **Classic Gameplay**: Control the snake to eat food and grow longer. Avoid hitting walls and yourself.
-   **Multiple Skins**: Choose from various themes including Classic, Neon, Golden, Ocean, Fire, and Matrix.
-   **Score Tracking**: Keep track of your current score and high score.
-   **Responsive Design**: Playable on desktop with keyboard controls.

## How to Play

1.  Open `index.html` in your browser.
2.  Click **Start Game** or press `Space`/`Enter`.
3.  Use the Arrow keys or `W`/`A`/`S`/`D` to control the snake.
4.  Eat the food to grow and increase your score.
5.  Avoid hitting the walls or your own tail.
6.  Press `Space` to pause/resume the game.

## Tech Stack

-   HTML5 Canvas
-   CSS3 (Animations, Flexbox)
-   JavaScript (ES6+)

## Project Structure

-   `index.html`: Main game structure and UI.
-   `style.css`: Styles for the game and UI elements.
-   `game.js`: Game logic, rendering, and input handling.
